Overview

The server solution (located in the ServerSolution directory) contains all the code needed to start the server that provides access to the business entities and their operations

Solution structure

 The server solution contains the following projects
  • BaseDesktop
  • BusinessLogic
  • Core
  • DataModel
  • EntityModel
