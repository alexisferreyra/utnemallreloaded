About the content of this folder

This folder of the solution contains the binaries of LayerD compilers.

In this sample application LayerD compilers are used for .NET platform when the server compiles Infrastructure Services for stores or the mall, also it is used for the compilation of assemblies when building Custom Services of stores/mall.

To generate .NET assemblies from high level services model we use custom DSL implemented with LayerD as compiler extensions - "classfactorys" on the terminology of LayerD framework. The used extensions are installed on the distributed compiler under "factorieslib" folder.

LayerD project is a framework with modular extensible compiler for multiple platforms - including .NET - and it is open source.

For more information on LayerD visit http://layerd.net.

For license information of these compilers read license.txt.

To get the source code of LayerD project visit http://sourceforge.net/projects/layerd.
