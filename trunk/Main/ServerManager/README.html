Overview

 The server manager is a WPF frontend that allows the user to interact with the server, with this application you will be able to manage:
  • Users
  • Customers
  • Categories
  • Services
  • Stores
  • Groups
  • Data models
This solution must be opened with Nicrosoft Expression Blend to see and modify the GUIs.

Solution structure

The ServerManager solution contains the following project references

  • BaseDesktop
  • EntityModel
  • LogicalLibrary
  • ServerManager
