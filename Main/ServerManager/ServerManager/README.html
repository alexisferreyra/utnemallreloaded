Overview

This project contains the components on WPF and the logic to interact with the server through the webservices that it exposes.

Key files
  • All the Add*.xaml.cs files: define a component to add and edit an entity for example AddUser allows to add and edit users
  • All the *Manager.xaml.cs files: define a component to list, filter and select an entity and select an operation (add/edit/delete)
  • Services.cs: defines a singleton for each proxy to a webservice that the server manager uses
  • List.xaml.cs: defines a generic list with three buttons to do operations on the objects that are listed
  • Tree.xaml.cs: defines a generic tree with three buttons to do ooperation on objects with a tree-like structure
